Documentação completa em inglês: [/docs/providers/spotify](/docs/providers/spotify).

## Configuração rápida

```ts
import Spotify from "vinextauth/providers/spotify";

Spotify({
  clientId: process.env.SPOTIFY_CLIENT_ID!,
  clientSecret: process.env.SPOTIFY_CLIENT_SECRET!,
})
```

URL de callback: `https://seudominio.com/api/auth/callback/spotify`
