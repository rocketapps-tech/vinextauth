**VinextAuth** é uma biblioteca de autenticação sem dependências construída sobre a Web Crypto API, projetada para rodar no edge do Cloudflare Workers.

É um substituto direto do NextAuth v4: mesmos callbacks, mesmo formato de sessão, mesmos nomes de cookies.

## Início rápido

```bash
npm install vinextauth
```

```ts title="src/auth.ts"
import VinextAuth from 'vinextauth';
import Google from 'vinextauth/providers/google';

export const { GET, POST, auth, toPages } = VinextAuth({
  secret: process.env.VINEXTAUTH_SECRET!,
  providers: [
    Google({
      clientId: process.env.GOOGLE_CLIENT_ID!,
      clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
    }),
  ],
});
```

## O que está incluído

- **12 providers OAuth** — Google, GitHub, Discord, Apple, Microsoft e mais
- **Magic link por e-mail** — com Resend ou qualquer API HTTP de e-mail
- **Credentials** — email/senha com rate limiting embutido
- **Adaptadores Cloudflare** — KV e D1 para sessões em banco de dados
- **Middleware edge** — proteja rotas antes de renderizar
- **React hooks** — `useSession`, `signIn`, `signOut`, `SessionProvider`
