Documentação completa em inglês: [/docs/providers/twitch](/docs/providers/twitch).

## Configuração rápida

```ts
import Twitch from 'vinextauth/providers/twitch';

Twitch({
  clientId: process.env.TWITCH_CLIENT_ID!,
  clientSecret: process.env.TWITCH_CLIENT_SECRET!,
});
```

URL de callback: `https://seudominio.com/api/auth/callback/twitch`
